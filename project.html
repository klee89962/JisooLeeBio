C - Magic Square Program

This program prints an n x n magic square (a square arrangement of the numbers 1, 2, ..., n2 in which the sum of the elements in any row, column, or diagonal is the same). This program should generate a magic square of a specified size n, where user will specify size at run time. Size must be an odd positive integer between 1 and 99.

# This program prints a n x n magic square where it is numbers from 1 to n^2 which sum up of elements in any row, coloum, or diagonal is the same. Size of magic square should be the size of n which is inputted by user. */

int main(void)
{
  int n;

  printf("Enter size of magic square: "); /* Prompts user value for size of magic square */
  scanf("%d", &n);

  while ((n % 2) == 0 || n >= 13) /* Checks if n is a valid size */
    {
      printf("Invalid size, try again...\n");
      printf("Enter size of magic square: ");
      scanf("%d", &n);
    }

  int magic_square[n][n], x = n / 2, y = 0, num = 2, x2, y2, i, j;

  for(i = 0; i != n; i++) /* Sets all the values in matrix equal to 0 */
    { 
      for(j = 0; j != n; j++){
    *( *(magic_square + i) + j) = 0;
      }
    }

  *( *(magic_square + y) + x) = 1; /* Sets the middle element of row 0 to 1 */

  while (num != ((n * n) + 1)) /* Checks if there is an attempt to go out of bound. So from row 0, next row won't be row -1, but it will be row n - 1 instead. For coloums, from col n - 1, next row wont be col n,
   but it will be col 0. Program also checks if array is empty (equals to 0). If it is empty, it fills the num in. If it is not empty, it puts the number directly below the previously stored number */
    {
      if (x == (n - 1))
    {
      x2 = x;
      x = 0;
    }
      else
    {
      x2 = x;
      x++;
    }
      if (y == 0)
    {
      y2 = 1;
      y = (n-1);
    }
      else
    {
      if(y == (n - 1))
        {
          y2 = 0;
          y--;
        }
      else
        {
          y2 = (y + 1);
          y--;
        }
    }

      if (*( *(magic_square + y) + x) != 0)
    {
      y = y2;
      x = x2;
    }

      *( *(magic_square + y) + x) = num;
      num++;
    }

  for(i = 0; i != n; i++) /* Prints all the values in array */
    {
      for(j = 0; j != n; j++){
        printf("\t%d", *( *(magic_square + i) + j));
      }
      printf("\n");
    }

}

C - Minimum Bill Program

This program determines the smallest number of $20, $10, $5, $2, and $1 billls/coins necessary to pay a dollar amount.

void pay_amount(int dollars, int *twenties, int *tens, int *fives, int *toonies, int *lonnie);

     "fives: %d\n"

VBA - Sudoku Solution Checker

This program checks the numbers on the cells for sudoku board to see if it is correct or not, and gives the rows, columns, and general range with problem if it is wrong.

The following code will check for the range for sudoku board. It will return boolean value based on the cell values.

Function checkRange(rng As Range) As Boolean

    Dim result As Boolean
    Dim found As Boolean

    result = True

    For i = 1 To rng.Count
        found = False

        For Each Cell In rng.Cells
            If Cell.Value = i Then
                found = True
                Exit For
            End If
        Next Cell

        If found = False Then
            result = False
            Exit For
        End If
    Next i

    checkRange = result

End Function

Using this function, every range of cells will need to be checked. Following function (checkAndFormat) will accept range of cells and check to see if range is valid.
